# IT-Support-Portfolio

## Introduction

This repository contains a collection of hands-on IT labs designed to simulate real-world enterprise IT environments. The labs focus on Active Directory administration, system troubleshooting, helpdesk workflows, and network configuration.

The goal of this project is to demonstrate practical skills used in IT support and system administration roles.

## Lab Environment

Virtualized infrastructure created using Oracle VirtualBox.

Environment includes:

- Windows Server 2022 Domain Controller
- Windows 10 Helpdesk Workstation (RSAT Tools)
- Windows 10 End User Machine
- Simulated enterprise network environment

## Core Skills Demonstrated

- Active Directory Administration
- User and Group Management
- Domain Joining Workstations
- Group Policy Management
- Helpdesk Troubleshooting
- Remote Administration
- File Shares and NTFS Permissions
- Software Deployment
- Ticketing System Workflow
